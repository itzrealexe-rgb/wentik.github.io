## 🚀 Features

- 🤖 **AI Moderation** — Smart filters for NSFW, spam, toxicity
- 📚 **File Library** — Upload, share, manage files with ratings
- 🎮 **Games** — Tic-Tac-Toe, Sea Battle, and more
- ⚡ **HTTP API** — Create custom endpoints on the fly
- 🔐 **Discord Login** — Secure authentication via Discord OAuth2
- 📊 **Dashboard** — Real-time server and user statistics
